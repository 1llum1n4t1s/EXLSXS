# Excel修復・最適化ツール

このWPFアプリケーションは、Excelファイル（.xlsx、.xlsm）の修復、最適化、パフォーマンス向上を行うツールです。

## 機能

### 1. ファイル修復
- **段階的修復レベル**: 安全な修復、標準修復、積極的修復から選択可能
- 破損したワークシートの修復
- 空のセルのクリーンアップ
- ファイル整合性のチェック
- Microsoft公式修復方法の実装

### 2. ファイル最適化
- 計算式の最適化
- スタイルの統合と重複削除
- 重複データの削除
- 未使用スタイルの安全な削除

### 3. パフォーマンス向上
- 計算モードの最適化
- 不要な計算式の削除
- 単純な計算式の定数化
- 画像の圧縮（オプション）

### 4. バッチ処理機能 🆕
- **フォルダ一括処理**: 複数のExcelファイルを一度に処理
- サブフォルダ対応
- 処理時間の推定
- 詳細な処理結果レポート

### 5. 詳細なログ機能 🆕
- **自動ログ出力**: 全処理過程を詳細に記録
- マイドキュメント内にログファイルを自動生成
- 処理完了後のログ表示オプション
- エラー追跡とデバッグ支援

### 6. 設定管理機能 🆕
- **設定の自動保存**: 前回の設定を自動で復元
- 最後に選択したフォルダの記憶
- カスタマイズ可能な処理オプション
- JSON形式での設定ファイル管理

### 7. バックアップ機能
- 元ファイルの自動バックアップ作成
- タイムスタンプ付きバックアップファイル
- バッチ処理時の専用バックアップフォルダ

## 使用方法

### 単一ファイル処理

1. **アプリケーションの起動**
   - ビルドして実行するか、実行可能ファイルを起動
   - 前回の設定が自動で復元されます

2. **ファイル選択**
   - 「ファイル選択」ボタンをクリック
   - 修復・最適化したいExcelファイル（.xlsx、.xlsm）を選択
   - 前回選択したフォルダが記憶されています

3. **修復レベル選択**
   - **安全な修復（推奨）**: データを保護しながら修復
   - **標準修復**: バランスの取れた修復
   - **積極的修復**: データ損失の可能性がありますが、より深刻な破損に対応

4. **オプション設定**
   - 実行したい処理のチェックボックスを選択
   - 設定は自動で保存され、次回起動時に復元されます

5. **処理実行**
   - 「実行」ボタンをクリック
   - プログレスバーで処理状況を確認

6. **完了**
   - 処理完了後、結果メッセージが表示されます
   - ログファイルの確認オプションが表示されます

### バッチ処理（複数ファイル一括処理）

1. **フォルダ選択**
   - 「フォルダ一括処理」ボタンをクリック
   - 処理したいExcelファイルが含まれるフォルダを選択

2. **サブフォルダ設定**
   - サブフォルダも含めて処理するかを選択

3. **処理確認**
   - 見つかったファイル数と推定処理時間が表示されます
   - 処理を開始するかを確認

4. **一括処理実行**
   - 各ファイルが順次処理されます
   - プログレスバーで全体の進捗を確認

5. **結果確認**
   - 成功・エラーファイル数の詳細レポートが表示されます

## 必要な環境

- .NET 9.0 以降
- Windows 10/11
- 十分なディスク容量（バックアップ用）

## 使用ライブラリ

- **EPPlus**: Excelファイルの読み書き
- **DocumentFormat.OpenXml**: OpenXML形式の操作
- **System.IO.Compression**: ファイル圧縮処理

## 注意事項

- 処理前に必ずバックアップが作成されます
- 大きなファイルの場合、処理に時間がかかる場合があります
- 破損が深刻なファイルは完全に修復できない場合があります
- 処理中はファイルを開かないでください

## ライセンス

このアプリケーションは非商用目的で使用できます。
EPPlusライブラリは非商用ライセンスで使用しています。

## トラブルシューティング

### よくある問題

1. **ファイルが開けない**
   - ファイルが他のアプリケーションで開かれていないか確認
   - ファイルの権限を確認

2. **処理が途中で止まる**
   - ファイルサイズが大きすぎる可能性があります
   - 十分なメモリがあるか確認

3. **エラーが発生する**
   - ファイルが破損している可能性があります
   - ログを確認して詳細なエラー内容を確認
