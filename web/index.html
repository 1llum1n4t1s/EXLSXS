Features

開いた瞬間、気持ちいいブックに。

Every sheet, neat the moment you open it.

シートごとにバラついた表示倍率・表示モード・カーソル位置を、リボンからまとめて整えます。 Zoom, view mode and cursor position — unified across the whole workbook, straight from the ribbon.

Apply to Every Sheet

全シートを一括整形

リボンで選んだ表示モードと表示倍率を、ブック内のすべてのシートへまとめて適用。一枚ずつ直す手間はもう要りません。 The view mode and zoom you pick in the ribbon are applied to every sheet in the workbook at once — no more fixing them one by one.

ズーム表示モード全シートOne-click
Unify Zoom

表示倍率をそろえる

シートごとにバラバラな拡大率を、指定したひとつの倍率に統一します。 Bring every sheet's zoom level to the single magnification you choose.

Unify View Mode

表示モードを統一

標準・ページレイアウト・改ページプレビューを、全シートで同じ表示にそろえます。 Normal, page layout or page-break preview — set the same view across all sheets.

Unify Fonts

フォントを統一

必要に応じて、ブック全体のフォントをそろえて見た目を整えます。 Optionally normalize fonts across the workbook for a consistent look.

A1 & First Sheet

A1 へ戻り、先頭シートへ

各シートの選択位置を A1 にリセットし、仕上げ後は一番左のシートをアクティブに。次に開く人が、いつも同じ場所から始められます。 Resets every sheet's selection to A1 and lands on the leftmost sheet, so the workbook always opens from the same spot.

Right in the Ribbon

リボンから直感操作

Excel のリボンに溶け込む UI。表示モードと倍率を選んで実行するだけ。覚えることは最小限です。 A UI that lives in the Excel ribbon — pick a view and zoom, then run. Almost nothing to learn.

Get started

選んで押すだけ。

Pick, press, done — here's all it takes.

3 ステップ Three steps

  1. 1
    表示を選ぶ Choose the view & zoom.
    リボンで表示モードと倍率、必要ならフォント統一を指定します。
  2. 2
    実行する Run it.
    ボタンひとつで、ブック内の全シートへまとめて適用します。
  3. 3
    整って完了 Tidied — cursor to A1, first sheet active.
    各シートのカーソルは A1 へ。仕上げると一番左のシートがアクティブになります。
